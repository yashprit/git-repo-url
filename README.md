> create github https or git-ssh url from username and repo


## Install

```sh
$ npm install --save git-repo-url
```


## Usage

```js
var git-repo-url = require('git-repo-url');

git-repo-url('Rainbow');
```

```sh
$ npm install --global git-repo-url
$ git-repo-url --help
```
